New String Features

We have several new feature inside of strings that are very helpful.

String interpolation

String concatenation is much easier with the new ES6 features. Variables can be injected into strings by using the ` (grave accent) instead of single/double quotes and using the ${x} symbols.

Can execute simple functions.

No loops .

Template Strings


function old() {
    const x = 'Hello';
    const y = 'World';

    console.log('Old: ' + x + ' ' + y);
}
old();


function newMethod() {
    const x = 'Hello';
    const y = 'World';

    console.log(`New: ${x} ${y}`);
}
newMethod();

Multiline Strings

You can also use the ` symbol for multi line strings.

Usage


function newMethod() {
    const x = 'Hello';
    const y = 'World';

    const template = `<div>
        <h1>${x}</h1>
        <h2>${y}</h2>
    </div>`
}
newMethod();
